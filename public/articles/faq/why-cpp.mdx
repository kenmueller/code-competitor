export const name = 'Why C++?'

All major code competitions support C++, winners of these competitions generally use C++ due to performance constraints given for solutions, C++ is a continuously improving language supporting low-level programming for hardware and performance critical software as well as high-level programming for large-scale systems, C++ is an industry standard for game development, compilers, robotics, and many other industries, all major operating systems, like Windows, macOS, and Linux are written in C/C++, and most major software, like Adobe Photoshop, Google Chrome, and Microsoft Office, is written in C++.
